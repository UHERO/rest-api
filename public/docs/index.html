Available Categories

GET /category

Tags: Categories

The category endpoint returns a list of available categories. If the id is given only the matching category will be returned. If the search_text is given only the categories matching the search terms will be shown
id

Category ID

 query number
search_text

Search Text

 query string
top_level

Show only Top Level Categories

 query boolean

Uses default content-types: application/json

200 OK

An array of categories

Example for application/json
{
    "categories": [
        {
            "id": 1,
            "name": "Summary"
        },
        {
            "id": 2,
            "name": "Income"
        },
        {
            "id": 3,
            "name": "Major Economic Indicator Summary",
            "parent": 1
        },
        {
            "id": 4,
            "name": "Summary of External Indicators",
            "parent": 1
        },
        {
            "id": 5,
            "name": "Personal Income Summary",
            "parent": 2
        }
    ]
}

Available transformations

GET /transforms

Tags: Transformations

Transformations supported by this api

Uses default content-types: application/json

200 OK

An array of transformations

Example for application/json
{
    "transformations": [
        {
            "description": "Value in levels",
            "formula": "x(t)",
            "id": "lvl"
        },
        {
            "description": "Change since last value",
            "formula": "x(t) - x(t-1)",
            "id": "chg"
        },
        {
            "description": "Percent change since last year",
            "formula": "(x(t)/x(t-n) - 1) * 100, where n is the number of observations in a year",
            "id": "pc1"
        }
    ]
}
